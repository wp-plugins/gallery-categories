Add categories of galleries to your WordPress website using the Gallery plugin by BestWebSoft.

== Description ==

This plugin allows creating different categories of galleries on your site. This way, you could organize the galleries on your website, which makes navigating through your content a whole lot easier. Please note that this plugin is an exclusive add-on to the Gallery Plugin by BestWebSoft and it will not work without it.

<a href="http://wordpress.org/plugins/gallery-categories/faq/" target="_blank">FAQ</a>
<a href="http://support.bestwebsoft.com" target="_blank">Support</a>

= Features =

* Actions: Adds all galleries, which the selected galleries was assigned to, to your page/post via the shortcode.  
* Actions: Assigns default gallery category to all galleries the moment the plugin is activated and when saving/updating the post. 
* Actions: When deleting a category, the galleries that belong to this category will not be deleted. Instead, these galleries will be moved to the category 
* Actions: An option to rename the default category.

== Installation == 

1. Upload the `gallery-categories` folder to `/wp-content/plugins/` directory.
2. Activate the plugin using the 'Plugins' menu in your WordPress admin panel.
3. Create a page or a post and insert the shortcode [print_gllr cat_id=*] into the text.

== Frequently Asked Questions ==

= How do I create a new gallery category? =

There are two ways you can create a new gallery category. First way: 

1. Similar to post categories, please click "Add new gallery category" when creating/editing a gallery. 
2. Enter a title for a new gallery category and press "Enter."

Alternately, you can do the following: 

1. Please go to Edit Gallery Categories page. 
2. Fill all the necessary fields and click "Add new gallery category." 

= Can I delete a default gallery category? =

No, you cannot delete it. All galleries should belong at least to one category. Default category option implies that new galleries, which do not have any category assigned to them, are automatically moved to default gallery category. 

= How can I change the title of default category? =

1. Please go to Edit Gallery Categories page. 
2. Click on the title of the default gallery category and enter the new one. 
3. Save the changes. 

= How can I view all galleries that belong to a certain category of galleries? =

Please paste the shortcode of the necessary gallery category to the page/post.
